# LLM annotation reports

Generated by the SPDB local LLM annotation pipeline.

| File | Description |
|------|-------------|
| `dry_run_report.md` | Prompt/validation dry run (`make llm-annotation-dry-run`) |
| `llm_annotation_report.md` | Summary of a local model run |
| `ollama_batch_failures.md` | Models skipped or failed during `make ollama-annotate-all` |
| `ollama_model_comparison.md` | Multi-model convergence/disagreement report (`make ollama-compare`) |
| `ollama_model_scores.csv` | Per-model label distributions |
| `ollama_pairwise_agreement.csv` | Pairwise model agreement and Cohen κ |
| `ollama_disagreement_units.csv` | Units with maximum cross-model disagreement |

Raw annotations: `data/experiments/llm_annotations/{MODEL_NAME}_pilot_100.jsonl`

Configure local models in `configs/ollama_models.yaml`.
