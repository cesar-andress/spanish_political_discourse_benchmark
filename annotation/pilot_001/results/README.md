# Pilot 001 agreement analysis results

Generated by `scripts/analysis/compute_*.py` and `generate_disagreement_report.py`.

## Inputs

| File | Description |
|------|-------------|
| `annotation/pilot_001/pilot_100_units_annotator_a.csv` | Annotator A export |
| `annotation/pilot_001/pilot_100_units_annotator_b.csv` | Annotator B export |

Both files must share the same `unit_id` rows and filled annotation columns.

## Outputs

| File | Description |
|------|-------------|
| `cohen_kappa.json` / `.md` | Overall κ and per-label agreement |
| `krippendorff_alpha.json` / `.md` | Overall α and per-label agreement |
| `confusion_matrix_*.csv` | Confusion matrices per dimension |
| `confusion_matrices.json` / `.md` | Combined confusion-matrix report |
| `disagreement_report.csv` / `.md` / `.json` | Unit-level disagreements |

## Reproduction

```bash
make pilot-agreement
```

Or run each script:

```bash
python -m scripts.analysis.compute_cohen_kappa
python -m scripts.analysis.compute_krippendorff_alpha
python -m scripts.analysis.compute_confusion_matrix
python -m scripts.analysis.generate_disagreement_report
```
